Helpers
#######

With the plugin Helpers you can seamlessly style various html elements. If any have been missed please feel free to create an issue on `bitbucket <https://bitbucket.org/lilHermit/cakephp-plugins-bootstrap4/issues>`_ and we'll endeavour to add them.

Styles
------

    - Form Elements, including submit buttons
    - Crumblists
    - Pagination
    - Buttons, Progress bars
    - Flash messages `error`, `info`, `warning` & `success` are styled as dismissable alerts
